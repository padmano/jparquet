## Getting Started


#### Install java 17
   ```bash
   brew install openjdk@17
   ```

#### Install Gradle (if not installed)
   ```bash 
   brew install gradle
   ```

#### Generate gradle wrapper
   ```bash 
   gradle wrapper
   ``` 

### Building

```bash
./gradlew build
```

### Running Tests

```bash
./gradlew test
```

### Running Examples

```bash
./gradlew run
